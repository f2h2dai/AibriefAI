Built for the live briefing cycle.

Pull from Hacker News, GitHub, arXiv, X/Twitter routes, and fallback sources on a scheduled cadence.

Step 02 Rank and summarize

Step 03 Publish the dashboard

Commit fresh JSON, health files, and briefing pages back to the repo so the site stays current.
